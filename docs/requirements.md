# mdedit 需求文档

## 1. 项目概述

mdedit 是一款轻量级、跨平台的 Markdown 编辑器，采用 Typora 式所见即所得（WYSIWYG）编辑模式，无需 WebView2 等浏览器内核依赖。

**技术栈**：Rust + egui + pulldown-cmark  
**目标平台**：Windows / macOS / Linux

## 2. 功能需求

### 2.1 文件操作

| 编号 | 功能 | 描述 | 优先级 |
|------|------|------|--------|
| F-001 | 新建文档 | 创建空白 Markdown 文档 | P0 |
| F-002 | 打开文档 | 通过文件对话框打开 .md 文件，支持 UTF-8 编码 | P0 |
| F-003 | 保存文档 | Ctrl+S 保存当前文档，保留原始 Markdown 格式 | P0 |
| F-004 | 另存为 | 将当前文档保存到新路径 | P0 |
| F-005 | 最近文件 | 记录最近打开的文件列表（最多 10 个） | P1 |

### 2.2 WYSIWYG 编辑

| 编号 | 功能 | 描述 | 优先级 |
|------|------|------|--------|
| F-010 | 实时渲染 | 编辑区直接渲染 Markdown 格式，光标所在块显示源码 | P0 |
| F-011 | 标题渲染 | H1-H6 按不同字号渲染 | P0 |
| F-012 | 行内格式 | 粗体、斜体、删除线、行内代码实时渲染 | P0 |
| F-013 | 列表渲染 | 有序/无序列表缩进渲染 | P0 |
| F-014 | 代码块 | 围栏代码块语法高亮显示 | P0 |
| F-015 | 引用块 | 块引用带左侧竖线样式渲染 | P0 |
| F-016 | 链接 | 渲染为可点击链接，点击用系统浏览器打开 | P1 |
| F-017 | 图片 | 渲染本地/网络图片（支持相对路径） | P1 |
| F-018 | 表格 | 表格渲染为网格样式 | P1 |
| F-019 | 分割线 | 水平分割线渲染 | P1 |

### 2.3 大纲视图

| 编号 | 功能 | 描述 | 优先级 |
|------|------|------|--------|
| F-020 | 大纲面板 | 左侧显示文档标题层级结构（H1-H6） | P0 |
| F-021 | 大纲导航 | 点击大纲项跳转到对应位置 | P0 |
| F-022 | 大纲折叠 | 支持折叠/展开子级标题 | P1 |
| F-023 | 大纲同步 | 编辑时大纲自动更新，当前位置高亮 | P1 |

### 2.4 编辑增强

| 编号 | 功能 | 描述 | 优先级 |
|------|------|------|--------|
| F-030 | 撤销/重做 | Ctrl+Z / Ctrl+Y，支持多步撤销 | P0 |
| F-031 | 查找替换 | Ctrl+F 查找，Ctrl+H 替换 | P1 |
| F-032 | 快捷键 | Ctrl+B 粗体、Ctrl+I 斜体等常用格式快捷键 | P0 |
| F-033 | 自动补全 | 输入 `- ` 自动续列表，输入 ``` 自动补全围栏 | P1 |

## 3. 非功能需求

| 编号 | 类别 | 要求 |
|------|------|------|
| NF-001 | 性能 | 冷启动时间 < 200ms |
| NF-002 | 性能 | 打开 5MB 文档内存占用 < 80MB |
| NF-003 | 性能 | 按键响应延迟 < 16ms（60fps） |
| NF-004 | 体积 | 单二进制文件 < 15MB |
| NF-005 | 兼容性 | 支持 CommonMark 规范 + GFM 扩展（表格、删除线、任务列表） |
| NF-006 | 可访问性 | 支持系统深色/浅色主题跟随 |
| NF-007 | 国际化 | UI 支持中英文，正确处理 CJK 文本换行 |

## 4. 用户场景

### 场景 1：日常笔记
用户打开 mdedit，新建文档，输入 `# 会议记录`，标题立即以大字号渲染。继续输入列表项，列表自动缩进。完成后 Ctrl+S 保存。

### 场景 2：阅读长文档
用户打开一个 5MB 的技术文档，左侧大纲面板显示完整目录结构。点击某个章节标题，编辑区跳转到对应位置。

### 场景 3：编辑代码文档
用户输入围栏代码块，代码自动语法高亮。光标移入代码块时显示原始 Markdown 标记，移出后恢复渲染视图。

## 5. MVP 范围（v0.1.0）

首个版本聚焦 P0 功能：
- 文件打开/保存
- WYSIWYG 渲染（标题、粗斜体、列表、代码块、引用）
- 大纲面板 + 点击跳转
- 基础编辑（撤销/重做、格式快捷键）
