# mdedit 测试用例文档

## 1. 文件操作

### TC-001 新建文档
- **前置条件**：应用已启动
- **操作**：Ctrl+N
- **预期**：编辑区清空，标题栏显示"未命名"，大纲面板为空

### TC-002 打开 Markdown 文件
- **前置条件**：存在 test.md 文件
- **操作**：Ctrl+O → 选择 test.md
- **预期**：文件内容加载到编辑区，大纲面板显示标题结构，标题栏显示文件名

### TC-003 保存文档
- **前置条件**：已打开文档并做了修改
- **操作**：Ctrl+S
- **预期**：文件写入磁盘，标题栏修改标记消失，内容为标准 Markdown 格式

### TC-004 保存未命名文档
- **前置条件**：新建文档，输入了内容
- **操作**：Ctrl+S
- **预期**：弹出保存对话框，选择路径后保存成功

### TC-005 打开大文件
- **前置条件**：存在 5MB 的 Markdown 文件
- **操作**：Ctrl+O → 选择该文件
- **预期**：文件在 2s 内加载完成，内存占用 < 80MB

## 2. WYSIWYG 渲染

### TC-010 标题渲染
- **操作**：输入 `# 一级标题` 后光标移到下一行
- **预期**：该行以 H1 大字号渲染，无 `#` 标记可见

### TC-011 光标进入已渲染块
- **前置条件**：文档中有已渲染的标题行
- **操作**：点击该标题行
- **预期**：该行切换为源码模式，显示 `# 一级标题`

### TC-012 粗体/斜体渲染
- **操作**：输入 `**粗体** *斜体*` 后光标移开
- **预期**：显示 **粗体** 和 *斜体* 样式，无星号可见

### TC-013 代码块语法高亮
- **操作**：输入围栏代码块（```rust ... ```）后光标移开
- **预期**：代码块带背景色显示，关键字有语法着色

### TC-014 列表渲染
- **操作**：输入 `- 项目一\n- 项目二` 后光标移开
- **预期**：显示为带圆点的无序列表，有缩进

### TC-015 引用块渲染
- **操作**：输入 `> 引用内容` 后光标移开
- **预期**：显示左侧竖线 + 缩进文本样式

### TC-016 分割线渲染
- **操作**：输入 `---` 后光标移开
- **预期**：显示为水平分割线

## 3. 大纲视图

### TC-020 大纲生成
- **前置条件**：文档包含 H1、H2、H3 标题
- **预期**：左侧大纲面板按层级显示所有标题

### TC-021 大纲点击跳转
- **操作**：点击大纲中某个标题
- **预期**：编辑区滚动到对应标题位置

### TC-022 大纲实时更新
- **操作**：在文档中新增一个 `## 新标题`
- **预期**：大纲面板立即出现该标题项

### TC-023 大纲当前位置高亮
- **操作**：在文档中滚动到某个章节
- **预期**：大纲面板中对应标题项高亮显示

## 4. 编辑功能

### TC-030 撤销
- **操作**：输入文本后按 Ctrl+Z
- **预期**：文本恢复到输入前状态

### TC-031 重做
- **操作**：撤销后按 Ctrl+Y
- **预期**：文本恢复到撤销前状态

### TC-032 快捷键格式化
- **操作**：选中文本后按 Ctrl+B
- **预期**：选中文本两侧添加 `**`，渲染为粗体

### TC-033 多步撤销
- **操作**：连续输入 3 次不同文本，按 3 次 Ctrl+Z
- **预期**：每次撤销恢复一步，最终回到初始状态

## 5. 性能测试

### TC-040 启动速度
- **操作**：冷启动应用
- **预期**：窗口显示时间 < 200ms

### TC-041 大文档滚动流畅度
- **前置条件**：打开 5MB 文档
- **操作**：快速滚动文档
- **预期**：帧率保持 60fps，无明显卡顿

### TC-042 输入延迟
- **前置条件**：打开 5MB 文档
- **操作**：在文档中间位置连续输入字符
- **预期**：每次按键到字符显示延迟 < 16ms
