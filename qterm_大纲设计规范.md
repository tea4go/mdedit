# 笔记大纲面板设计规范

## 1. 面板整体布局

| 属性 | 值 |
|------|-----|
| 面板宽度 | ~260px |
| 面板背景色 | `#1E1E1E`（深灰色，类 VS Code 暗色主题） |
| 面板内边距 | 0（标题栏以下无额外内边距） |
| 面板右边框 | 1px `#333333` 分隔线，与右侧编辑区隔开 |

## 2. 顶部标题栏

| 属性 | 值 |
|------|-----|
| 高度 | ~36px |
| 背景色 | 略浅于面板背景，约 `#252526` |
| 文字内容 | "笔记大纲" |
| 字体大小 | 12-13px |
| 字体粗细 | normal（400） |
| 字体颜色 | `#CCCCCC`（浅灰色） |
| 左侧内边距 | 12px |
| 右侧操作区 | 可能有折叠/关闭图标按钮，颜色 `#888888`，hover 时变亮 |

## 3. 大纲列表结构

### 缩进规则（每级递增）

| 层级 | 缩进 | 图标 |
|------|------|------|
| H1 | 12px 左内边距 | 小实心圆点 `●` 或短横线，颜色 `#569CD6`（蓝色） |
| H2 | 24px 左内边距 | 小空心圆点 `○` 或短横线，颜色 `#4EC9B0`（青绿色） |
| H3 | 36px 左内边距 | 小菱形 `◆` 或短横线，颜色 `#DCDCAA`（黄色） |
| H4 | 48px 左内边距 | 小横线 `–` 或短横线，颜色 `#C586C0`（紫色） |

> **说明**：截图中的图标是小型标记符号，不同级别用不同颜色区分。每级缩进约增加 12-14px。

### 条目样式

| 属性 | 值 |
|------|-----|
| 字体大小 | 13px（统一，不分层级） |
| 字体颜色 | `#D4D4D4`（浅灰白色） |
| 行高 | ~28px（每条目高度） |
| 条目间距 | 0（紧凑排列，无额外间距） |
| 文字截断 | 超长标题截断，末尾省略号 `...`，不换行 |
| 图标与文字间距 | 6px |

### 选中/高亮状态

| 属性 | 值 |
|------|-----|
| 背景 | `#04395E`（深蓝色半透明高亮） |
| 左边框指示器 | 2px 宽 `#007ACC` 蓝色竖线（可选） |
| 文字颜色 | `#FFFFFF`（纯白，比未选中更亮） |

### Hover 状态

| 属性 | 值 |
|------|-----|
| 背景 | `#2A2D2E`（微亮灰色） |

### 展开/折叠

| 属性 | 值 |
|------|-----|
| 箭头图标 | 小三角形 `▶`/`▼`，8px 大小 |
| 箭头颜色 | `#888888` |
| 箭头位置 | 条目文本左侧，图标右边 |
| 点击行为 | 点击箭头折叠/展开子级；点击文本跳转到对应标题 |

## 4. 滚动条

| 属性 | 值 |
|------|-----|
| 显示策略 | 自动隐藏（内容溢出时出现） |
| 宽度 | 6px |
| 滑块颜色 | `#555555`（hover 时 `#777777`） |
| 轨道颜色 | 透明 |

## 5. 颜色方案汇总

| 用途 | 色值 |
|------|------|
| 面板背景 | `#1E1E1E` |
| 标题栏背景 | `#252526` |
| 默认文字 | `#D4D4D4` |
| 暗淡文字 | `#888888` |
| 标题栏文字 | `#CCCCCC` |
| 选中项背景 | `#04395E` |
| Hover 背景 | `#2A2D2E` |
| H1 图标色 | `#569CD6` |
| H2 图标色 | `#4EC9B0` |
| H3 图标色 | `#DCDCAA` |
| H4 图标色 | `#C586C0` |
| 分隔线 | `#333333` |
| 强调蓝 | `#007ACC` |

## 6. 交互行为

1. **单击条目**：编辑区滚动到对应标题位置
2. **单击箭头**：折叠/展开该标题下的子级
3. **双击条目**：无特殊行为（等同单击）
4. **鼠标悬停**：显示完整标题文本（tooltip），适合被截断的长标题
5. **键盘导航**：上下键移动选中项，Enter 跳转，左右键折叠/展开

## 7. 实现参考（组件结构）

```
Panel (width: 260px, bg: #1E1E1E)
├── HeaderBar (height: 36px, bg: #252526)
│   ├── Text "笔记大纲" (color: #CCCCCC, 13px)
│   └── [可选] CloseButton (color: #888888)
└── ScrollArea (auto scrollbar)
    └── OutlineTree
        └── OutlineItem (每条标题)
            ├── IndentSpacer (12px * level)
            ├── ExpandArrow (▶/▼, color: #888888, 8px)
            ├── LevelIcon (颜色随层级变化)
            └── TitleText (13px, truncate with ellipsis)
```
