# WhaleTerminal 笔记界面设计规范（本地文档）

> 本文档定义 WhaleTerminal 笔记模块中**本地文档**相关的完整 UI 设计规范。覆盖布局、尺寸、颜色来源、交互行为。供 Rust 原生程序复刻界面使用。

---

## 1. 整体布局

### 1.1 三栏布局

```
┌───┬──────────┬───┬─────────────────────────────────────┐
│   │  左侧面板 │   │                                     │
│ R │ ┌──────┐ │ R │         右侧编辑区                   │
│ i │ │ Tab栏 │ │ e │                                     │
│ b │ ├──────┤ │ s │  ┌─────────────────────────────────┐│
│ b │ │      │ │ i │  │         工具栏 (35px)           ││
│ o │ │ 文件 │ │ z │  ├─────────────────────────────────┤│
│ n │ │ 树 / │ │ e │  │                                 ││
│   │ │ 大纲 │ │   │  │     编辑器区域                    ││
│   │ │ /搜索│ │   │  │     (IR / SV / WYSIWYG)            ││
│   │ │      │ │   │  │                                 ││
│   │ │      │ │   │  │                                 ││
│   │ ├──────┤ │   │  │                                 ││

│   │ └──────┘ │   │                                     │
└───┴──────────┴───┴─────────────────────────────────────┘
```

### 1.2 面板尺寸

| 区域 | 尺寸 | 来源 |
|------|------|------|
| 侧边栏 (Ribbon) | `leftNavWidth` px | 系统配置 |
| 左侧面板宽度 | 可拖拽调整 | `leftNoteWidth` 配置值 |
| 左侧面板最小 | `preferenceState.defaultConfig.leftNoteWidth` | 各模块默认值 |

| 右侧编辑区 | flex: 1 (占满剩余) | — |
| 分割线宽度 | 2px (hover 3px) | ResizeableWrapper |

## 9. Rust 实现建议

### 9.1 编辑器选型

| 方案 | 说明 | 推荐度 |
|------|------|--------|
| **原生实现** | 纯 Rust + GPU 渲染，完全可控 | 高（工作量大） |

### 9.2 关键模块划分

```
src/note/
├── mod.rs
├── editor/
│   ├── mod.rs
│   ├── toolbar.rs          // 工具栏按钮定义与事件
│   ├── mode_switch.rs      // IR/SV/WYSIWYG 模式切换
│   ├── auto_save.rs        // 自动保存逻辑
│   ├── scroll.rs           // 滚动位置管理
│   └── zoom.rs             // 字体缩放
├── sidebar/
│   ├── mod.rs
│   ├── file_tree.rs        // 文件树 (MDTree)
│   ├── outline.rs          // 大纲
│   ├── search.rs           // 搜索结果树
│   └── toolbar.rs          // 底部工具栏
├── search_bar.rs           // 搜索/替换栏
├── color_selector.rs       // 颜色选择器
└── settings.rs             // 笔记设置
```

### 9.3 关键实现注意事项

1. **文件树虚拟滚动**: 大量文件时必须启用虚拟滚动，行高固定 `fontSize * 3 - 4`
2. **拖拽自动展开**: 悬停 800ms 触发，需要管理定时器 Map
3. **大纲渲染**: 从文档解析标题层级，手动插入编号
4. **搜索高亮**: 在编辑器中实现文本搜索与高亮标记
5. **表格宽度**: 编辑器宽度变化时重新计算 `80%` 值
6. **自动保存**: debounce 模式，最小 60 秒，静默无 UI 提示
7. **图片预览**: 全屏 overlay + 滚轮缩放（步长 0.1x）
8. **Base64 过滤**: 粘贴时必须过滤，防止文档体积膨胀
